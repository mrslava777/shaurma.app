# Шаурма на Левкова v3 — Новый дизайн

Дизайн как в приложении доставки еды:
- Светлая тема, красные кнопки
- Карточки товаров с фото
- Нижнее меню навигации
- Категории с горизонтальным скроллом
- Боковое меню (гамбургер)
- Модалка выбора пива снизу

## Deploy
1. Загрузите на GitHub
2. render.com → New + → Web Service
3. Build: `pip install -r requirements.txt`
4. Start: `gunicorn app:app`
